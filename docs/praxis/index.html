Praxis

Liberation theology's methodological keyword: theology as 'critical reflection on praxis' (Gutiérrez) — starting from concrete practice among the poor rather than from doctrine applied downward. The 1984 Instruction's critique targeted versions of this method it judged to import Marxist analysis uncritically.[1]
